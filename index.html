shrimp.css is a open source css framework for markdown generated blog pages styling

Usage

  1. Create an HTML document with a basic structure
    2.                 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My blog</title>
</head>
<body>

</body>
</html>
  2. Download the shrimp.css file
    3. Download Link
  3. Add the shrimp.css file to your project with the link tag
    4. <link rel="stylesheet" href="shrimp.css"/>
  4. Write some html inside your html body tag
    5.                 
<main>

<figure>
    <img src="public/img/wide.jpg" alt="photo-of-mountain-under-cloudy-sky" srcset=""/>
</figure>
